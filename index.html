§1 — Прилагательные от существительных

Суффикс -Н-Одна Н
✦
Основа на одну согласную + суффикс -Н- = одна Н
зима → зимний
⚠️
Основа уже оканчивается на -Н- + суффикс -Н- = две НН
осень → осеннийлуна → лунный
Суффиксы -АН- / -ЯН- / -ИН-Одна Н
✦
Перед пропуском видишь А / Я / И → всегда одна Н
лебедь → лебединый серебро → серебряный кожа → кожаный
⚠️ Исключения (НН!): стеклянный, оловянный, деревянный
Суффиксы -ЕНН- / -ОНН-Две НН
💡
Найди исходное слово → выдели корень → всё после корня = суффикс. Если суффикс начинается на Е или О → пиши НН
✦
авиация → корень АВИАЦИ- → суффикс начинается на О → авиациОННый
революция → революциОННыйторжество → торжестВЕННый
⚠️ Исключение: ветреный (но безветренный)
Алгоритм для прилагательных от существительных
1
Найди исходное существительное
2
Перед пропуском стоит А/Я/И? → одна Н (кроме стеклянный, оловянный, деревянный)
3
Перед пропуском стоит Е/О? → две НН
4
Основа на -Н- + суффикс -Н-? → две НН

§2 — Производные предлоги: слитно или раздельно?

ВСЛЕДСТВИЕ // В СЛЕДСТВИИ
✦
Слитно + Е = «из-за»
вследствие болезнивследствие засухи
✦
Раздельно + И = следственные действия
в следствии вскрылись подробности
НАВСТРЕЧУ // НА ВСТРЕЧУ
✦
Слитно = «по направлению к»
навстречу ветру
✦
Раздельно = предлог + сущ.; между ними можно вставить слово

ВВИДУ // В ВИДУ
✦
Слитно = «из-за / по причине»
ввиду опоздания
✦
Раздельно = «на виду»
в виду всей Москвы
⚠️ иметь в виду — фразеологизм, всегда раздельно
НАСЧЁТ // НА СЧЁТ
✦
Слитно = «об(о)»
насчёт денег
✦
Раздельно = предлог + сущ. «счёт»; можно вставить слово
на мой счёт в банке
⚠️ за счёт (благодаря) — всегда раздельно
НАПОДОБИЕ // НА ПОДОБИЕ
✦
Слитно = «вроде»
наподобие бабочки
✦
Раздельно = предлог + сущ. «подобие»
смотрел на подобие человека
НЕСМОТРЯ НА // НЕ СМОТРЯ НА
✦
Слитно = «вопреки»
несмотря на дождьневзирая на слабость
✦
Раздельно = деепричастие «не имея зрительного контакта»
шёл, не смотря под ноги
ОТТОГО // ОТ ТОГО
✦
Слитно = «потому»
проснулся оттого, что стучали
✦
Раздельно = «того» указывает на конкретное
от того дома можно дойти

§3 — Союзы и частицы

ЧТОБЫ // ЧТО БЫ
✦
Слитно = союз цели (нельзя убрать БЫ)
хочу, чтобы ты ушёл
✦
Раздельно = местоим. + частица; БЫ можно убрать
не знал, что бы мне купить
Тест: убери БЫ — смысл сохранился? → раздельно
ТОЖЕ / ТАКЖЕ // ТО ЖЕ / ТАК ЖЕ
✦
Слитно = взаимозаменяемы и = союзу «и»
я тоже = я также = и я люблю квас
✦
Раздельно = ЖЕ можно убрать, указание на что-то
сделал так же, как вчерато же, что вчера
ЗАТО // ЗА ТО
✦
Слитно = можно заменить на «но»
глупая, зато красивая
✦
Раздельно = «то» указывает на конкретный предмет
спрятался за то дерево

§1 — НЕ пишется раздельно

Противопоставление с А
↔
Есть союз А, слова взаимоисключают друг друга
не большая, а уютная не друг, а враг не хороший, а плохой
Слова-усилители
💬
Рядом стоят отнюдь, далеко, вовсе — оттягивают НЕ от любой части речи
отнюдь не красивый далеко не смело вовсе не красавец
НИ-слова
💬
Рядом: нисколько, ничуть, нимало, никак, никому, никогда
ничуть не правильный нисколько не уставший никому не известный
Краткие причастия
✦
С краткими причастиями — всегда раздельно
работа не выполнена смена не закончена
Глаголы и деепричастия
✦
С глаголами и деепричастиями — всегда раздельно
не видел не слышал не думая
⚠️ Кроме слов, которые без НЕ не употребляются: невзлюбил, ненавидел, негодовать
Местоимения, числительные, союзы, частицы
✦
С местоимениями, числительными, союзами и частицами — раздельно
не я не пять не то

§2 — НЕ пишется слитно

Без НЕ не употребляется
✦
Слово без НЕ не существует → пишем слитно
невзлюбить ненавидеть негодовать нелепый небрежно
Можно заменить синонимом без НЕ
↔
Прилагательные, существительные, наречия — если есть синоним
нехороший = плохой неправда = ложь нехорошо = плохо
💡 Нет противопоставления с А + можно заменить синонимом → пишем слитно

§3 — Исключения

Слитно, хотя кажется раздельным
⚡
Неопределённые местоимения — слитно
некто нечто некогда некоторый
⚡
Частица нежели и союз несмотря на то что — слитно
лучше, нежели раньше несмотря на то что опаздывал
Алгоритм — как решать задание 13
1
Есть слова отнюдь / далеко / вовсе / ни-слова? → раздельно
2
Есть противопоставление с А? → раздельно
3
Это глагол / деепричастие / краткое причастие? → раздельно
4
Слово без НЕ не существует? → слитно
5
Можно заменить синонимом без НЕ? → слитно

§1 — Спряжения глаголов

I и II спряжениеОпределяется по инфинитиву
I
I спряжение — все глаголы не на -ИТЬ
писАТЬ, читАТЬ, лаЯТЬ
Окончания: -Е-, -У-, -Ю-
пишУ / пишЕМ пишЕШЬ / пишЕТЕ пишЕТ / пишУТ
II
II спряжение — все глаголы на -ИТЬ

Окончания: -И-, -А-, -Я-
говорЮ / говорИМ говорИШЬ / говорИТЕ говорИТ / говорЯТ
💡 Спряжение определяй только по инфинитиву (неопределённой форме), когда окончание безударное

§2 — Исключения

I спряжениеХотя должны быть во II
⚡
Глаголы не на -ИТЬ, но с окончаниями I спряжения (Е, У, Ю)
брИТЬ → бреЕШЬ, бреЮТ стелИТЬ → стелЕШЬ, стелЮТ зиждИТЬся зыбИТЬся
II спряжениеХотя оканчиваются не на -ИТЬ
⚡
11 глаголов с окончаниями II спряжения (И, А, Я)
гнАТЬ дышАТЬ терпЕТЬ обидЕТЬ слышАТЬ видЕТЬ ненавидЕТЬ держАТЬ смотрЕТЬ вертЕТЬ зависЕТЬ
💡 Запомни: гнать, дышать, терпеть, обидеть, слышать, видеть, ненавидеть, держать, смотреть, вертеть, зависеть

Задание 14 — Слитно/раздельно/дефис

Главные правила
Всегда раздельнов течение, в продолжение, в заключение, за счёт
Всегда дефис-то, -либо, -нибудь, кое- → что-то, кое-кто
Производные предлоги
вследствиеСлитно+Е = «из-за». Раздельно+И = следствие
навстречуСлитно = «по направлению к». Раздельно = на [важную] встречу
ввидуСлитно = «из-за». Раздельно = «на виду». Иметь в виду — всегда раздельно
насчётСлитно = «об». Раздельно = на [мой] счёт. За счёт — всегда раздельно
наподобиеСлитно = «вроде». Раздельно = на подобие [чего-то]
несмотря наСлитно = «вопреки». Раздельно = деепричастие (не смотря под ноги)
оттогоСлитно = «потому». Раздельно = от того дома (указание)
Союзы и частицы
чтобыСлитно = союз цели. Раздельно = БЫ можно убрать
тоже / такжеСлитно = = «и». Раздельно = ЖЕ можно убрать
затоСлитно = «но». Раздельно = за то дерево (указание)
Слитно (наречия)
Запомнивдребезги, наугад, навсегда, наспех, зачастую, вплотную, надолго, изредка, напролом, вдвое, сбоку
Раздельно
Запомнив упор, до смерти, на ходу, под силу, по двое, за границей, как будто, вряд ли, как раз, всё равно
